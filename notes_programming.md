# Programming Notes

## Beginner

- Programming is the practice of writing instructions for computers.
- Python is a popular language for data engineering because it is easy to read and has strong library support.
- Key Python building blocks:
  - variables
  - data types
  - expressions and operators
  - functions and control flow

## Intermediate

- Object-oriented programming (OOP):
  - classes encapsulate state and behavior
  - objects are instances of classes
  - use inheritance and composition to share behavior
- NumPy for numerical computing:
  - arrays provide fast numeric operations
  - vectorized computations are much faster than loops
- Pandas for data manipulation:
  - `DataFrame` and `Series` are core structures
  - use `read_csv`, `groupby`, `merge`, and `pivot_table`
- PySpark for big data processing:
  - distributed dataframes with Spark SQL
  - use transformations such as `filter`, `map`, and `reduce`
- Advanced Python features:
  - decorators, context managers, generators
  - iterators and lazy evaluation

## Advanced

- Testing:
  - write unit tests with `pytest`
  - use test fixtures and assertions
  - mock external dependencies when needed
- Performance optimization:
  - profile code to find bottlenecks
  - avoid repeated expensive operations
  - use efficient data structures and algorithms
- Best practices:
  - write readable, maintainable code
  - follow PEP 8 style guidelines
  - document functions and modules clearly
